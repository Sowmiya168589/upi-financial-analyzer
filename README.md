An AI-powered Streamlit app that analyzes UPI PDF transactions (Paytm, GPay, PhonePe) to provide **personalized financial insights**.  

- **Extracts & structures transactions** from PDFs  
- **Categorizes spending**:Junk Food, Shopping, Entertainment, Travel, Bills, Grocery, Others  
- **Detects wasteful spending**: movies, junk food, impulse shopping  
- **Generates AI-based recommendations** for budgeting and savings  
- **Interactive charts** for easy financial overview  

Built with **Python, NLP, Streamlit**, and **LLM-style reasoning**. Deployable on **Hugging Face Spaces**.
